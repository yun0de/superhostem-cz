Quick start

How to add your property

After the first login, adding a property is the main step that unlocks further work in the application. Once the first property is established, you can follow up on it by importing bookings, guest forms and other settings.

Help Center Previous step Next step →
What awaits you next?

You choose the method of establishing the property and then in its settings you will go in three subsequent steps: fees to the municipality, Ubyport data and finally iCal connection.

Where do you get to it?

Via the Real Estate section

Open in the left menu Real estate. If still you have nothing established, you will see an empty status You have no properties yet and button Add first property.

Via onboarding on Overview

On the page Overview you will see a block for the first settings real estate. You can fill in directly there Property name a Airbnb iCal URL.

What you will see in the Real Estate section

Empty state after first entry

  • button Synchronize all properties
  • button Add property
  • information You have no properties yet
  • button Add first property
  • link How to set up automatic forwarding
After the establishment of the property, it is up to you to set it up. That's where he gives the point is to continue straight through the three mandatory areas: fees to the municipality, Ubyport data a iCal.

How to proceed step by step

Step 1

Open Properties or Onboarding

If you want to go the classic route, click on in the menu Real estate and use the button Add property or Add first property.

If you already have an iCal from Airbnb, you can stay on the page Overview and use the onboarding form.

Step 2

Prepare the basic data

Prepare at least before proceeding property name. Ideally, you should also prepare three pieces of information, which you will fill in immediately after in the property settings.

  • fees to the municipality
  • Ubyport ID and password
  • Airbnb iCal URL

Step 3

Create a property

In onboarding after filling in the fields Property name a Airbnb iCal URL you use the button Create a property and import reservations.

If you don't have iCal yet, first set up the property via Real estate and immediately after that go to hers settings where you will continue with the next three steps.

Step 4

Fill out the fees to the municipality

The first related part of the settings is the information k fee to the municipality. A must have for every property clear, if you have already registered, where do you submit the fees and how you will send them.

This step is also important so that you then at the end of the month correctly generated the report that is sent to the city.

Don't have these details? How about fees to the municipality

Step 5

Fill in the Ubyport ID and password

Once you have resolved the municipal fees, proceed to Ubyport data. These are important for follow-up on guest reporting and other administrative part of operation.

If you don't have them on hand now, it's a good idea to prepare them as soon as possible, so that the property settings remain complete. If you haven't already Ubyport established at all, first you need to go through registration and only then add the obtained login data to Superhostem.

  • Ubyport ID
  • Ubyport password
Don't have a Ubyport account or login? How to register for Ubyport

Step 6

Finally, connect iCal

The last follow-up part is iCal connection, which ensures loading of reservations and synchronization of the calendar. Thanks to that it will start filling in the reservation overview and the application will start working with stays.

If you're using Airbnb or another OTA source, insert the appropriate iCal URL right in this property settings section.
